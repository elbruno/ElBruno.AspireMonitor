## Learnings

### Image Generation Strategy

1. **NuGet Icons:**
   - Primary icon (256x256): High visibility, must be clear at small sizes
   - Secondary icon (128x128): Even more compressed, must still be readable
   - Style: Modern, professional, related to monitoring/dashboard

2. **Promotional Graphics:**
   - LinkedIn (1200x630): Show app in action, feature highlights
   - Twitter (1024x512): Tech-forward, eye-catching
   - Blog (1200x630): Educational, show use case

3. **Demo GIF:**
   - Capture live app (system tray interaction, status colors, URL clicking, config)
   - 10-15 seconds (short, attention-holding)
   - Show: icon in tray → click to show window → show status colors → click URL → open browser → open config
